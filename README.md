# dotnet-weather-api
🚀 A simple Weather API built using ASP.NET Core that fetches real-time weather data from a third-party API (Visual Crossing). This project demonstrates API integration, caching with Redis, error handling, and environment variable management.
Features:

✅ Fetch real-time weather data by city name 🌍
✅ Caching with Redis to reduce API calls and improve performance
✅ Uses environment variables for API keys and configurations
✅ Error handling for invalid cities or API failures
✅ Scalable and extendable with .NET Core Web API
Technologies Used:

    C# (ASP.NET Core Web API)
    RestSharp (for HTTP requests)
    Redis (for caching)
    Microsoft.Extensions.Configuration (for environment variables)
    Postman/Swagger (for testing)
